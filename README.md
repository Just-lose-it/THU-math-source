# THU-math-source
一些积攒下来的清华基础数学课程的资源，当前涵盖微积分、线性代数、概率统计类课程。主要收集较新的、当前教学体系下的资源，过古早的资源一般无太大参考价值，暂不考虑上传。

**使用资源前建议先阅读各学科的README.md**

欢迎大家纠错和补充。如果需要更多类型的资源，也欢迎大家提出。
